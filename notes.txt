class - 2
==========

    -> How To Use Service Now Platform ?

        -> We Need Service Now Instance 

    
    -> What is an Instance (Cloud Term)

        -> An instance in cloud computing is a virtual server (virtual machine) 
        running on a cloud provider's physical infrastructure, 
        acting as a dedicated resource for running applications, databases, or websites.

    -> What is an Instance (Service Now)

        -> When you are accessing ServiceNow, you are accessing an instance of ServiceNow platform.

        -> An Organization can have multiple instances 

            -> Development Instance     -   New Features Development 
            -> Test Instance            -   Test the Features Developed (QA - CHECK PASSED)
            -> Production Instance      -   Live Data (Customers)

            -> NOTE: Each of the above instance is a single implementation of ServiceNow Platform.
        
        
        -> NOTE: Your data is protected 

            -> Your Organization Data isn't mingled in the same database as another Organization's data 

        -> NOTE: Easy To Do Data Migration

            -> Each instance has customer data that can be exchanged between other instance.
            For example it is easy for data to be migrated from a Development instance to QA instance to Production instance.

        -> NOTE: Instance Upgrades 

            -> Zurich Version To Australia Version 

            -> Upgrades are made on individual instances.

            -> Upgrades are made on Non-Production instance (QA)

class - 3
==========

    -> Working With Lists 

        -> List in servicenow shows "Multiple Records" from a single table
        (Such as incident table, user table, problems table etc)

        -> Each row = One Record (Single Incident i.e INC0000005 whole row)

        -> Each Column = One Field (Number,Short description, Priority)

        -> Each Cell = One Field for that record (INC0000009 in the Number Column, New in the State Column)

    
    -> Interactions With Lists 

        -> On any list, users can open a record 

            -> Click any blue value (often this will be number or name) it opens Form for that record 

        -> Sort the list 

            -> Click on column header 
            -> Click one to sort in ascending order and Click again to sort in descending order 
        
        -> Grouping Option 

            -> Group Multiple Records of same type under one set 

            -> Select column 

        -> Change / Update Field In List 

            -> Double Click on the field 

        -> Personalize List 

            -> Click On Gear Icon
